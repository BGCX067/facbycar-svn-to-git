Les services

Comment s'inscrire ?

Pour s'inscrire, accédez à la page en cliquant sur le bouton de gauche "s'inscrire". On vous demande alors de choisir un login ainsi qu'un mot de passe.

Que faire si je ne trouve aucun trajet ?

Nous vous proposons d'élargir votre recherche aux villes alentours. Ainsi sélectionnez les kilomètres du périmètre.
Vous pouvez aussi entrer une autre ville de départ et/ou d'arrivée.
Si vous ne trouvez aucune réponse pertinente, nous vous invitons à ajouter un trajet en tant que passager.

Comment est calculé le coût du trajet ?

Le conducteur fixe lui-même le prix par passager sachant que la voiture sera complète. Ainsi, si la voiture n'est pas complète, le prix sera plus élevé.

Comment se désinscrire?

Dans votre "Espace personnel", à la rubrique "Modifier mes informations" pour trouverez un bouton "se desinscrire". Pour confirmer cette action, il faudra entrer votre mot de passe.

Les membres

Comment contacter un passager ou un conducteur ?

Vous êtes sur la page de résultat de recherche. Cliquez sur le bouton "Contacter" pour obtenir plus de précsions sur le trajet qu'il propose.

Comment voir les évaluations laissés par mes contacts ?

Les derniers commentaires et évaluations sont affichés en bas de la page "Espace personnel". Vous pouvez cliquez sur "Voir tous les commentaires".

Comment noter un membre ?

En allant sur le profil d'un membre, vous pouvez laisser un commentaire ainsi qu'une note.

Mes trajets

Comment s'inscrire à un trajet ?

Vous êtes sur la page de résultat de recherche. Cliquez sur le bouton "Participer au trajet". Vous devez attendre que le membre valide votre demande pour être sûr de faire partie du trajet.

Comment se désinscrire d'un trajet ?

Dans votre "Espace personnel" vous pouvez voir le trajets auquels vous êtes inscrits sur la colonne de droite. Cliquez sur "Annuler ma demande" après avoir pris soin de contacter le conducteur suffisamment tôt (une semaine).

Comment accepter une demande de trajet ?

Les demandes de trajets s'affichent dans l' "espace perso" à droite dans la liste "Mes trajets comme conducteur". Le nombre de demandes s'affichent, cliquez pour pouvoir contacter le membre, voir son profil, voir ses évaluations ou accepter directement sa demande.
Il y a aussi un accès rapide sur la colonne de gauche "Demandes en attente".

Modifier mes informations

Comment modifier les informations de mon compte (age, sexe, email, numéro de téléphone, voiture) ?

Rendez-vous sur la page "espace perso", puis cliquez sur "Editer mon profil" ou "Afficher mes voitures", vous accéderez à un formulaire à valider par la suite.

Comment rajouter une voiture ?

Rendez-vous sur la page "Espace personnel", puis cliquez sur "Ajouter une voiture". Vous pouvez mettre une photo de votre voiture.

Comment changer un de mes trajets en tant que conducteur ?

Vous devez tout d'abord vous assurer que personne n'est insrit à votre trajet. Si ce n'est pas le cas, veuillez contacter les passagers le plus tôt possible (une semaine avant) pour les informer des changements, au cas où le trajet ne les conviendrais plus.
Ensuite, dans votre "Espace personnel", accédez à "modifier ce trajet".

Comment supprimer un de mes trajets en tant que conducteur ?

Vous devez tout d'abord vous assurer que personne n'est insrit à votre trajet. Si ce n'est pas le cas, veuillez contacter les passagers le plus tôt possible (une semaine avant) pour qu'ils puissent trouver un autre trajet.
Ensuite, dans votre "Espace personnel", cliquez sur "supprimer ce trajet".

Comment préciser mon indisponibilité sur un trajet régulier ?

Lors de l'ajout d'un trajet, vous pouvez préciser la date de début et de fin de validité.
